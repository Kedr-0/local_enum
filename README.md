Local Enumeration es el proceso de recolección sistemática de información de un sistema al que ya se tiene **ACCESO LOCAL**, 
con el objetivo de identificar configuraciones, servicios y permisos, con el objetivo de hallar vectores potenciales para escaladar privilegios.
